# 隐私政策

**生效日期:** 2026年3月29日

**应用名称:** Zuan

**开发者:** 2026 Antelope Inc.

---

## 介绍

您的隐私对我们至关重要。本隐私政策解释了 Zuan（“本应用”）如何收集、使用和保护您的信息。我们致力于确保您的数据保持私密且安全。

---

## 数据收集与处理

### 设备端处理

Zuan 在您的设备上完全执行所有 AI 转录和音频处理。**没有任何音频数据、转录内容或个人内容会被发送到外部服务器进行处理。** 您的内容始终保留在您的设备中。

### 我们不收集的数据

- 音频录音或文件
- 转录内容
- 听力历史记录
- 个人消息或通讯
- 位置数据
- 联系信息
- 浏览记录

### 存储在您设备本地的数据

以下数据存储在您的设备本地，绝不会传输到我们的服务器：

- **播客订阅和剧集元数据** — 存储在设备上，并通过 Apple iCloud (CloudKit) 在您的各台设备之间同步。
- **已下载的播客剧集** — 存储在您的设备本地存储空间中。
- **AI 生成的转录内容** — 与缓存的媒体一起存储在本地。
- **播放位置和偏好设置** — 使用标准的 iOS 存储机制在本地存储。
- **导入的音频文件** — 存储在应用的本地 Documents 目录中。

### iCloud 同步

Zuan 使用 Apple 的 CloudKit 在您的各台设备之间同步播客订阅。此数据由 Apple 管理，并受 [Apple 隐私政策](https://www.apple.com/legal/privacy/)的约束。我们不为此同步运行任何中间服务器。

---

## 第三方服务

### 播客搜索

Zuan 使用 iTunes Search API 和/或 Podcast Index API 来搜索播客。当您搜索播客时，您的搜索查询会被发送到这些服务以返回结果。这些请求中不包含个人信息。

### 广告

Zuan 使用 Google AdMob 显示广告。AdMob 可能按照 [Google 隐私政策](https://policies.google.com/privacy)中的描述收集并使用数据。这可能包括：

- 设备标识符
- IP 地址
- 大致位置（国家/地区级别）
- 广告交互数据

您可以通过设备的隐私设置控制个性化广告：

1. 在您的 iOS 设备上前往 **设置 > 隐私与安全性 > 跟踪**。
2. 禁用“允许 App 请求跟踪”以限制广告个性化。

Zuan 遵守 Apple 的应用跟踪透明度 (ATT) 框架，并会在启用个性化广告前请求您的同意。

### 分析

Zuan 使用 Google Firebase Analytics 收集匿名使用数据，帮助我们改进应用。此数据**不与您的身份相关联**，包括：

- 功能使用事件（例如：开始播放、订阅播客、导入媒体）
- 应用会话信息（例如：会话时长、应用版本）
- 常规设备信息（例如：设备型号、操作系统版本、国家）

Firebase Analytics **不**收集：

- 您的姓名、电子邮件或任何个人标识符
- 您的转录内容或音频内容
- 特定的听力历史记录或播放内容

此数据由 Google 处理，并受 [Google 隐私政策](https://policies.google.com/privacy)的约束。您可以前往 [Google 合作伙伴网站政策](https://policies.google.com/technologies/partner-sites)了解有关 Google 如何使用这些数据的更多信息。

### RSS 摘要

当您订阅播客时，Zuan 会直接从发布者的服务器获取播客的 RSS 摘要。这是一项标准的 HTTP 请求，可能会向播客托管方公开您的 IP 地址，正如任何播客应用一样。

## 数据保留

由于所有用户数据都存储在您的设备本地，您对数据拥有完全控制权：

- 通过卸载应用**删除应用数据**。
- 在应用内**删除具体项目**（转录内容、下载内容、库项目）。
- 使用“设置”中的存储管理功能**清除所有缓存数据**。

---

## 儿童隐私

Zuan 不会有意收集 13 岁以下儿童的个人信息。本应用不要求创建帐户，也不收集个人数据。
